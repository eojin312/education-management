# 배경
* 스프링MVC를 이용한 토이프로젝트 진행이 필요했다
* 이 프로젝트를 진행하던 시점은 고등학생이었다
* 학교나 교육청에서 필요한 시스템인 나이스에 대해 구상해보게 되었다
* 학사관리를 축소한 토이프로젝트를 스프링MVC기반으로 만들기로 결심했다
